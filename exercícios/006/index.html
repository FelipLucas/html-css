Principais Formatações 🙂

Negrito/Destaque

Nessa frase veremos uma termo em negrito

para fazer o negrito usamos o código <b>(Não semântica).

Nessa frase veremos uma termo em destaque

para fazer o destaque usamos o código <strong>(Semântica).

Itálico/Ênfase

Nesta frase veremos um termo em itálico

para fazer o itálico usamos o código <i>(Não semântico).

Nesta frase veremos um termo em ênfase para fazer o ênfase usamos o código <em>(Semântico)

Marcado

Nesta frase veremos um termo marcado para fazer o marcado usamos o código <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno. Pequeno:<small>. Grande:<big>

Texto deletado

Podems marcar um texto como excluído, indicando que ele pode ser lido mas não considerado,
para isso usamos o código<del>.

Texto inserido

Podemos marcar um texto como inserido, para dar uma ênfase e indicar que le foi adicionado depois,
para isso usamos o código<ins>.

Texto sobscritos

Para insegir coisas tipo x2O+3 usamos o código <sup>

Textos subscritos

Para inserir coisas tipo H2O usamos o código <sub>.
